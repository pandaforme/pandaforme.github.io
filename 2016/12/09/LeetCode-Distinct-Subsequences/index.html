Distinct Subsequences


Given a string S and a string T, count the number of distinct subsequences of T in S.

A subsequence of a string is a new string which is formed from the original string by deleting some (can be none) of the characters without disturbing the relative positions of the remaining characters. (ie, “ACE” is a subsequence of “ABCDE” while “AEC” is not).

Here is an example:
S = “rabbbit”, T = “rabbit”

Return 3.

解法：

剛看到題目一直搞不清楚count the number of distinct subsequences of T in S這句話的涵義…，後來找到這篇number of distinct subsequence解釋了這句話。原來是我誤解了，題目是想知道S有多少個子字串等於T。

假設$S = (s_1, s_2, …, s_m)$，$T = (t_1, t_2, …, t_n)$。
若$s_m == t_n$，則$S = (s_1, s_2, …, s_m-1)$，$T = (t_1, t_2, …, t_n-1)$為子問題，$S = (s_1, s_2, …, s_m-1)$，$T = (t_1, t_2, …, t_n)$也為子問題。

剛開始只有列出第一項，測了幾個test case結果總是少幾個。後來才想到第二項要考慮進去，例如：S=”cbaa”， T=”ba”。

若$s_m != t_n$，則$S = (s_1, s_2, …, s_m-1)$，$T = (t_1, t_2, …, t_n)$為子問題。

這樣就可以透過以上的遞迴式寫程式了，但是當字元過多時後就會爆掉或是很慢，可以改寫成Dynamic programming的版本。

宣告一個二維陣列matrix[T.length() + 1][S.length() + 1]，matrix[i][j]代表$S = (s_1, s_2, …, s_j)$，$T = (t_1, t_2, …, t_i)$的解。
若$s_j == t_i$，則matrix[i][j] = matrix[i - 1][j - 1] + matrix[i][j - 1]。
若$s_j != t_i$，則matrix[i][j] = matrix[i][j - 1]。
